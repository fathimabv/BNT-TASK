CSS Units: em, rem, px, %

In CSS, units like em, rem, px, and % are used to control font size, spacing, and Layouts.Example

PX (Pixels):
Font size is 20px (fixed size).
Width is 200px — does not change with parent or screen.
EM:
Font size is 1.5em = 1.5 × 16px = 24px.
Width is 20em = 20 × 24px = ~480px.
REM:
Font size is 1.5rem = 1.5 × root 16px = 24px.
Width is 20rem = 20 × 16px = 320px.
% (Percentage):
Font size is 150% of the parent (16px × 1.5 = 24px).
Width is 50% of the parent container.
